# dcbd-buttons

> DcBD buttons

[![NPM](https://img.shields.io/npm/v/dcbd-buttons.svg)](https://www.npmjs.com/package/dcbd-buttons) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dcbd-buttons
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'dcbd-buttons'
import 'dcbd-buttons/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
